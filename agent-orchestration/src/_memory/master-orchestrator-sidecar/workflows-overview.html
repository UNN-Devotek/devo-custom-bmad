All workflows + 6 Tracks + Review Track · left-to-right · tmux split-pane labeled · in-process fallback documented

tmux mode detection When tmux is active ($TMUX is set): split-pane agents spawn in new panes as shown.
▶ Fallback (no tmux): All split-pane steps automatically run in-process instead — all agents share the same conversation context. Review gates already run in-process; quick-dev fix steps run sequentially in the same conversation.
